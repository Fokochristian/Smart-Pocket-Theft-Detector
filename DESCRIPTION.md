# Smart-Pocket-Theft-Detector
Smart Pocket Theft Detector is an Android application developed in Kotlin that uses the phone’s built-in accelerometer and gyroscope sensors to detect suspicious movement patterns associated with phone theft. The app monitors sudden acceleration spikes and rapid orientation changes that typically occur when a device is forcefully removed from a user’s pocket. When such abnormal motion is detected, the application immediately triggers an alert to warn the user. This project aims to address the common problem of smartphone theft in crowded or public environments by providing an additional layer of real-time motion-based security.
